Step 2: Load the Dataset
1. Click Home Tab
2. Click Get Data
3. Select Text/CSV

4. Browse and select:
Week1_Basic_Sales_Data.csv
5. Click Load

Step 3: Verify Data in Data View

1. Click the column
2. Go to Column Tools
3. Change Data Type accordingly:
○ Date → Date
○ Sales → Decimal Number
○ Product/Category/Region → Text
